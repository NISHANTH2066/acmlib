Długość wykresu funkcji $f : [a, b] \to \mathbb{R}$:
  \hfill
  $\int_a^b \sqrt{f'(x)^2 + 1} \; dx$.

Pole figury obrotowej $f : [a, b] \to \mathbb{R}$:
  \hfill
  $2 \pi \int_a^b |f(x)| \sqrt{f'(x)^2 + 1} \; dx$.

Objętość figury obrotowej $f : [a, b] \to \mathbb{R}$:
  \hfill
  $\pi \int_a^b f(x)^2 \; dx$.
